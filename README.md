# Nghia Van Vo Academic Website

This repository contains the source for `https://NghiaVo99.github.io`, an academic website built with the Academic Pages Jekyll template.

## Local Preview

Docker is the recommended local build path on this machine:

```bash
docker compose up
```

Then open `http://localhost:4000`.

## Content

Primary site content lives in:

* `_pages/` for home, awards, CV, publications, projects, and teaching pages
* `_publications/` for publication entries
* `_portfolio/` for project entries
* `_teaching/` for teaching entries
* `files/Nghia_Van_Vo_CV.pdf` for the public downloadable CV

The private source CV PDF is intentionally ignored by Git.
